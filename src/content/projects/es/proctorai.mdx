Mi proyecto de maestría centrado en la supervisión automatizada con IA, <a href="/academic-writings/dissertation/proctoring-dissertation.pdf" target="_blank" rel="noopener noreferrer" className="font-medium text-black underline underline-offset-4 hover:text-gray-600 dark:text-white dark:hover:text-gray-300">Un sistema de examen basado en la web mejorado mediante supervisión automatizada con detección de actividad en segundo plano</a>.

Esta aplicación fue creada para garantizar la integridad académica durante los exámenes remotos. Al aprovechar las tecnologías web modernas junto con algoritmos avanzados de aprendizaje automático, el sistema puede monitorear y marcar actividades inusuales en tiempo real. 

El desarrollo de este proyecto me impulsó a expandir mis habilidades más allá del desarrollo web estándar, integrando una lógica backend compleja con una experiencia frontend fluida y accesible construida en Next.js.
